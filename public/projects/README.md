# Screenshots des Projets

Placez vos screenshots dans ce dossier avec les noms suivants :

## Fichiers requis pour la page d'accueil :

- `hellojade.jpg` - Screenshot principal de HelloJADE (affiché sur la page d'accueil)
- `easyrecruit.jpg` - Screenshot principal de EasyRecruit (affiché sur la page d'accueil)
- `helloluna.jpg` - Screenshot principal de HelloLuna (affiché sur la page d'accueil)
- `rbbc.jpg` - Screenshot desktop de RBBC Website (affiché sur la page projets)

## Fichiers requis pour les pages détaillées :

- `hellojade-main.jpg` - Screenshot en haute résolution pour la page détaillée HelloJADE
- `easyrecruit-main.jpg` - Screenshot en haute résolution pour la page détaillée EasyRecruit
- `helloluna-main.jpg` - Screenshot en haute résolution pour la page détaillée HelloLuna
- `rbbc-main.jpg` - Screenshot desktop pour la page détaillée RBBC

## Format recommandé :

- **Résolution** : Minimum 1920x1080px, idéalement 2560x1440px pour les screenshots principaux
- **Format** : JPG ou WebP (WebP recommandé pour de meilleures performances)
- **Qualité** : Optimisée pour le web (qualité 80-85% pour JPG)
- **Aspect ratio** : 16:9 ou 21:9 pour un rendu optimal

## ⚠️ IMPORTANT - Screenshot Mobile RBBC :

Pour afficher le design responsive de RBBC, vous devez ajouter :
- `rbbc-mobile.jpg` - Screenshot de la version mobile/portrait du site RBBC

Ce screenshot sera affiché côte à côte avec la version desktop pour mettre en avant le design responsive.

## Fonctionnement :

- **Page d'accueil** : Affiche 2 screenshots par projet avec effet stack (2ème image apparaît au hover)
- **RBBC spécial** : Affiche Desktop + Mobile côte à côte avec badge "Responsive Design"
- **Pages détaillées** : Affichent le screenshot principal en haut de page
